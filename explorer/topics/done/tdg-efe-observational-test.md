# Topic: Confronting the EFE Prediction with Tidal Dwarf Galaxy Data

## Question
The EFE numerical test predicts Synchronism gives sigma ~ 10.5-14.5 km/s for a 10^7 M_sun TDG at g_ext = 1.0 a0, while MOND gives sigma ~ 10.9-40.9 km/s. Can this be tested against the NGC 5291 system (Bournaud et al. 2007, Lelli et al. 2015)?

## Context
The explorer's EFE finding (efe-numerical-test-results.md) identified the TDG test as the cleanest discriminator between Synchronism and MOND. This is now on the MOND Unification page as a novel prediction. But the prediction has not been confronted with actual TDG data.

## Why It Matters
This would be the first zero-cost test that could genuinely distinguish Synchronism from MOND. The NGC 5291 system has three TDGs with measured rotation curves and known external field. If the prediction survives, it's a genuine novel result. If it fails, the bounded C (Hill form) is falsified.

## Suggested Starting Points
- Lelli et al. 2015: TDG rotation curves in NGC 5291
- Bournaud et al. 2007: original identification
- The explorer's work/efe_numerical_test.py code
- Compute sigma predictions for actual NGC 5291 TDG parameters (mass, radius, g_ext)
