## Major Discoveries

### 1. The Self-Consistency Gap
Session #66 writes C = tanh(βzJC), then identifies βzJ = γ·log(ρ/ρ_crit + 1). Substituting gives C = tanh(γ·log(ρ/ρ_crit + 1)·**C**) — but the framework drops the C factor and uses the explicit form. The site's "self-consistency" framing is overstated.

### 2. Tanh Is a Theorem, Not a Choice
Six independent derivation routes (Weiss, Brillouin J=1/2, MaxEnt binary, binary MMSE, BCS, Boltzmann machines) all produce tanh from the same mechanism: two competing states + Boltzmann weights. Systems with Bose statistics, polynomial self-consistency, or >2 states produce different functions. The site should frame this as "a mathematical theorem" — much stronger than "arises from constraints."

### 3. The EFE Loophole
The archive explicitly denies the EFE, but this denial rests on algebraic reasoning that ignores the field equation structure. The nonlinear Poisson equation couples internal and external fields through C's nonlinearity — exactly the mechanism that generates MOND's EFE in AQUAL. Nobody has numerically tested whether Synchronism predicts an EFE through this channel.

### 4. Gamma Direction Bug (5 files affected)
The authoritative source (equations.ts) says large γ = quantum. Three interactive tools (coherence-explorer, gamma-calculator, phase-boundary-visualizer) plus the equation-walkthrough have it reversed. The tools visitors actually touch teach the wrong physics.

### 5. γ = 2/√N_corr Failed Empirically
Session #395 tested against SPARC data: amplitudes ~700% too large, 0% improvement in scatter. This isn't on the site.
